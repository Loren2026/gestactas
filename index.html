Especificaciones Técnicas del MVP

Borrador v1.0 — 27 de abril de 2026
Arquitectura del MVP
Frontend:

React PWA + Vite + PWA Plugin

Almacenamiento:

IndexedDB (Sin servidor, datos privados)

Transcripción:

OpenAI Whisper (0,36€/h) o Web Speech API (gratis)

Generación Actas:

Anthropic Claude API (0,03€/acta)

Flujo de Trabajo (6 pasos)
1
Crear Comunidad

Formulario con dirección, propietarios, plantilla

2
Crear Junta

Doble convocatoria +30 min, asistentes, orden del día

3
Grabar Junta

MediaRecorder API, waveform, marcadores

4
Transcribir

Whisper (nube) o Web Speech (local)

5
Generar Acta con IA

Claude API genera acta legal

6
Exportar a Word

docx-js genera .docx en navegador

Base de Datos IndexedDB
comunidades Nombre, dirección, CIF, propietarios, plantilla
propietarios Datos personales, contacto, propiedad, cargo
juntas Tipo, fecha, hora, asistentes, orden del día
grabaciones Archivo de audio, duración, fecha
transcripciones Texto, método, idioma, marcas, fecha
actas Contenido, .docx, generada por, fecha
